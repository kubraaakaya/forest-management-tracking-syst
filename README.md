# 🌲 Forest Management Tracking System

## Project Overview

Forest Management Tracking System is a web-based application developed using PHP and MySQL. The system enables users to manage forest and tree records through a secure authentication mechanism and perform CRUD (Create, Read, Update, Delete) operations.

The project was developed as part of a Web-Based Programming course to gain practical experience in backend development, database management, and web application design.

## 🚀 Features

* User Registration and Login
* Session Management
* Add New Tree Records
* View Existing Records
* Update Tree Information
* Delete Records
* MySQL Database Integration
* Responsive Web Interface

## 🛠 Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap

## 📂 Project Structure

```text
config/          -> Database configuration files
public/          -> Application pages and assets
database.sql     -> Database schema and sample data
```

## ⚙️ Installation

1. Clone the repository.
2. Import the `database.sql` file into MySQL.
3. Configure the database connection settings.
4. Run the project using XAMPP, WAMP, or a similar local server environment.
5. Open the application in your browser.
